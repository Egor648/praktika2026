# Frontend - Система управления закупками

React приложение для работы с контрагентами и лотами.

## Установка
```bash
cd demo-frontend
npm install
```

## Запуск
```bash
npm start
```

Приложение откроется на http://localhost:3000

## Требования

- Node.js 16+
- npm 8+
- Запущенный backend на http://localhost:8080

## Функции

- Просмотр списка контрагентов и лотов
- Добавление новых записей
- Редактирование существующих записей
- Удаление записей
- Фильтрация по полям
- Сортировка по столбцам
- Навигация между разделами

## Технологии

- React 18
- Consta Design System
- Axios
